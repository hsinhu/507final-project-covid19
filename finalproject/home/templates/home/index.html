Welcome to Xin Hu's COVID19 Visualization Website

There are 4 visualization options for this website.

World Cases

This page shows today's world map of COVID19. User can select confirmed cases, deaths and active cases. The color for each country will be different corresponding to their cases.

Explore!
US State Cases

This page shows today's US map of COVID19. User can select confirmed cases, deaths and per capita confirmed cases and per capita death cases. The color for each state will also be different corresponding to their cases.

Explore!
US Projection

This page shows projected daily needed medical resources, daily death number and total death number from covid19 start to future for selected state by users. The default is the whole US.
